# gpgram3
GPT3 Telegram bot assistant

## Installation

```sh
pip install -r requirements.txt
```

Create `.env` with:

```
BOT_TOKEN    = <TELEGRAM_TOKEN>
API_USER     = <OPENAI_USER>
API_PASSWORD = <OPENAI_PASSWORD>
```

## Run

```sh
python server.py
```

```sh
python bot.py
```
